Detailed Description

Reflective material implementing perfect mirror reflection.

Creates mirror-like surfaces that reflect incoming rays according to the law of reflection. The reflected ray direction is computed as: r = d - 2(d·n)n

Parameters
reflectivityHow reflective the surface is (0 = no reflection, 1 = perfect mirror)
colorTint applied to reflected light (RGB 0-255)

[Indirect Lighting] Determines if and how light bounces off the surface.

This is called once per hit to handle "Passive" lighting like mirrors, glass, or matte reflections. If it returns true, the Core will trace the 'scattered' ray.

Parameters
ray_inThe incoming ray that hit the surface.
recordThe data from the hit.
attenuation[Out] How much the reflected light is filtered by the material's color.
scattered[Out] The new ray produced by the bounce.
Returns
true if the ray continues (reflection/refraction), false if it stops (absorbed).
