[Indirect Lighting] Determines if and how light bounces off the surface.

This is called once per hit to handle "Passive" lighting like mirrors, glass, or matte reflections. If it returns true, the Core will trace the 'scattered' ray.

Parameters
ray_inThe incoming ray that hit the surface.
recordThe data from the hit.
attenuation[Out] How much the reflected light is filtered by the material's color.
scattered[Out] The new ray produced by the bounce.
Returns
true if the ray continues (reflection/refraction), false if it stops (absorbed).

Implements IMaterial.

[Direct Lighting] Determines how the surface reacts to a specific light source.

This is called by the Core for EVERY visible light in the scene. It handles the "Active" lighting (the bright spots and the base color).

Parameters
recordThe data from the hit (normal, point, etc.).
lightDirThe direction vector pointing from the hit point TO the light.
lightColorThe color and intensity of the light source.
Returns
The Vec3 color contribution (how much this light brightens the pixel).
